# FlowBridge
FlowBridge dispatches irrigation controller events to external hardware.

A modular bridge that listens to and Irrigation Controller's events and routes 
station state changes to configurable output backends, enabling integration 
with arbitrary hardware or control systems.

## Supported Irrigation Controllers

 * [OpenSrpinkler](https://opensprinkler.com/) - [MQTT](https://openthings.freshdesk.com/support/solutions/articles/5000859089-how-to-use-mqtt)

## Supported External Hardware

 * [Waveshare's Industrial Modbus RTU Relay Modules](https://www.waveshare.com/product/modules/others/power-relays/modbus-rtu-relay-b.htm)
